# STACK.md

## Technology Stack

### Frontend
- **Framework**: React 18 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

### Backend / BaaS
- **Platform**: Appwrite
- **SDK**: `appwrite` (Web SDK), `node-appwrite` (Server SDK)

### Monitoring Sidecar
- **Runtime**: Node.js 20 (Alpine)
- **Framework**: Express
- **Library**: `dockerode` (Docker API client)

### Infrastructure
- **Platform**: Docker / Docker Compose
- **Arch**: ARM64 (Ubuntu 24.04)
- **Proxy**: Traefik (recommended)
